About Dungeon Tools

It looks like it's your first time here. Welcome to Dungeon Tools!

Thank you for using Dungeon Tools!

Dungeon Tools is a local web application that provides support and enhancements for table top role playing games, for both casual players as well as game masters. As a local web application, all data is stored in your browser, and will persist so long as your browser allows. (For most purposes, this should be as long as you don't uninstall it, though you may encounter data loss in private or incognito tabs.) There is no backend server that Dungeon Tools communicates with, and your data never leaves your computer. This also means that Dungeon Tools does not ever store backups of your data, so it is up to you to make sure you keep things safe.

Dungeon Tools is not a supplement to any particular game. We do note endorse stealing any content, especially copyrighted content and intellectual property, that belongs to other game developers. Although this application will contain all the tools necessary to recreate many rules, tables, and features found in other games, please use these for your own purposes and do not share that data outside of Dungeon Tools. If you wish to use Dungeon Tools alongside your favorite game, please consider supporting that game by purchasing content from the game before importing data into your Dungeon Tools storage.
